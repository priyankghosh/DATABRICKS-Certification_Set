Write a SQL query that will add a table that may or may not already exists called managed_table. It is a csv file. The data is located at the parth "url" for the URL. Add the options for it being formatted and copied. (1-2)
DROP TABLE IF EXISTS managed_table;
CREATE TABLE managed_table;
COPY INTO managed_table
FROM 'url'
FILEFORMAT = csv
FORMAT_OPTIONS ('mergeSchema' = 'true',
'delimeter' = ',',
'header' = 'true')
COPY_OPTIONS ('mergeSchema'='true');
